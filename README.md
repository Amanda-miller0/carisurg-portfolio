# CariSurg Portfolio

## What is this project and what does it do?

This repository contains coursework and project materials completed as part of the CariSurg MedTech Pathways Programme. It includes notebooks, documentation, and supporting files related to machine learning, healthcare analytics, and AI-assisted clinical decision support.

## Who is it for?

This repository is intended for instructors, mentors, and fellow participants in the CariSurg MedTech Pathways Programme. It may also be useful for anyone interested in healthcare artificial intelligence and machine learning applications.

## How do I install and run it?

1. Clone the repository:

```bash
git clone https://github.com/<your-username>/carisurg-portfolio.git
```

2. Navigate to the project folder:

```bash
cd carisurg-portfolio
```

3. Install the required libraries:

```bash
pip install -r requirements.txt
```

4. Open the Jupyter notebooks located in the `notebooks/` directory.

## Where does the data come from?

Datasets used throughout the programme are stored in the `data/` folder. The Week 0 dataset and any future datasets are intended to be placed there. Data sources are documented within the corresponding notebooks and reports.

## Repository Structure

* `notebooks/` – Jupyter notebooks and analyses.
* `docs/` – Memoranda, proposals, and supporting documentation.
* `data/` – Datasets and data descriptions.
* `src/` – Source code and scripts.
* `requirements.txt` – Python package dependencies.
* `README.md` – Project overview and usage instructions.
